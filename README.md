# my-vim

## how to install my-vim config

```
cd ~
git clone git@github.com:newpanderking/my-vim.git .vim
ln -s .vim/vimrc .vimrc
```
done!

> 说明：vim配置是一个不断定制化的过程，先定制自一部分自己的配置，有一部分是从网上copy过来的。不断的打磨自己顺手的编辑器。


## 命令说明

### F1~5 说明
- F1: 内置功能键，帮助文档；
- F2: 行号是否显示, 行号开关；  
- F3: nerd tree 目录开关；
- F4: 是否折叠行，折行开关；
- F5: 粘贴模式开关；
- F6: 语法高亮开关；

- 全选`, sg`

### markdown 语法命令
- <C-P>: markdown 预览，浏览器预览
